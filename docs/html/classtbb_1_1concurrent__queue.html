Detailed Description

template<typename T>
class tbb::concurrent_queue< T >

An bad implementation of a concurrent (thread-safe) queue that wraps std::queue with locks. Useful as a (rather naive) alternative to tbb::concurrent_queue. This is what you get when you don't use TBB.

Reference: https://www.justsoftwaresolutions.co.uk/threading/implementing-a-thread-safe-queue-using-condition-variables.html
